# Admin

Use this section to perform platform-level administration in a safe order.

## Who this is for

Admins who manage accounts, committees, memberships, and sign-on group rules.

## Before you start

1. Confirm you are using an admin account.
2. Decide whether your current task is login/dashboard, setup, or membership/rule updates.

## Step-by-step

1. Start with [Admin Login and Dashboard](/docs/02-admin/01-admin-login-and-dashboard) to enter the admin area.
2. Continue with [Accounts and Committee Management](/docs/02-admin/02-accounts-and-committee-management) to create accounts/committees and assign users.
3. Use [Committee Memberships and Sign-On Group Rules](/docs/02-admin/03-committee-memberships-and-oauth-rules) for membership updates and sign-on group rules.

## What you should see

- A clear sequence from access, to setup, to membership and rule updates.
- Fewer permission and setup mistakes by following the page order.

## If something goes wrong

- You cannot access admin pages:
  Reconfirm you are signed in with an admin account.
- A task feels out of place:
  Return to this index and open the matching page in the sequence above.
