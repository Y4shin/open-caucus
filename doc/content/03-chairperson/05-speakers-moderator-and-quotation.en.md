# Speakers Moderator and Quotation

Use this page to keep speaking order clear and fair while the meeting is running.

## Who this is for

Chairpersons who manage the live speakers queue and speaking rules.

## Before you start

1. Open the meeting moderation page.
2. Make sure the current agenda point is active.
3. Confirm attendees are already in the meeting.
4. Decide your default speaking rules before long debate starts:
   - **FLINTA* quotation**
   - **First-speaker bonus**
   - **Moderator**

## Layout on desktop vs mobile

- Desktop:
  You can see **Speakers List** and **Add Speaker** next to each other while you work.
- Mobile:
  Sections are stacked. Finish one task (for example add a speaker), then scroll to the next section.

## Manage the Live Speakers Queue

### Step-by-step

1. Add a speaker:
   - In **Add Speaker**, search by attendee name or number.
   - Click **Add regular speech** or **Add Point of Order (PO) speech**.
   - Use **Add Point of Order (PO) speech** only for special procedural interventions used by your committee.
2. Start the next person:
   - Click **Start Next Speaker**.
3. End the current speech:
   - Click **End Speech** when the person is done.
4. Adjust order when needed:
   - Use **Give Priority** on a waiting speaker.
   - Use **Remove Priority** to undo.
5. Remove wrong entries:
   - Click **Remove** on the speaker row and confirm.
6. Re-center long lists:
   - Click **Scroll To Active Position** if you lose the current position.

### What you should see

- Added speakers appear immediately in **Speakers List**.
- Only one person is marked as currently speaking.
- Priority changes move waiting speakers in the visible order.
- Completed speeches stay visible with their speaking duration (mm:ss) shown next to their name.

## Configure Moderator and Quotation Rules

### Meeting-level settings

1. Open the **Settings** tab and stay on **Meeting**.
2. Set **FLINTA* quotation** to enabled or disabled. When enabled, the queue alternates between FLINTA* and non-FLINTA* speakers (round-robin), with FLINTA* going first. Within each group, first-speaker bonus and request time determine order.
3. Set **First-speaker bonus** to enabled or disabled.
4. Choose a **Moderator** from the attendee list, or keep `-- none --`.

### Agenda-point overrides

1. In **Settings**, switch to **Agenda Point**.
2. For **FLINTA* quotation** and **First-speaker bonus**, choose:
   - **Inherit from meeting**
   - enabled
   - disabled
3. Optionally set an **Agenda Point Moderator**.

### What to confirm after updates

- The selected values remain set after the automatic update.
- The status lines show the expected `On`/`Off` result.
- Agenda-point overrides apply only to the currently active agenda point.

## If something goes wrong

- You cannot manage speakers:
  Check whether an agenda point is active.
- A person is missing in **Add Speaker**:
  Confirm the attendee has joined this meeting.
- A speaker button is disabled:
  That person already has that speech type waiting.
- Mobile flow feels confusing:
  Complete one section first, then scroll to the next section.
- Wrong speaker started:
  End the current speech, then start the correct next speaker.
