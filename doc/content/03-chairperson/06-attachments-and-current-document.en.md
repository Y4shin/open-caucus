# Attachments and Current Document

![Agenda tools attachments](../../assets/captures/app-agenda-tools-attachments.en.light.desktop.png)

Use this page to upload files for the active agenda point and control which file attendees should read now.

## Who this is for

Chairpersons who share documents during a live meeting.

## Before you start

1. Open the meeting moderation page.
2. Activate the correct agenda point.
3. Open the **Tools** tab.
4. In **Attachments**, click **Manage Files**.
5. Keep the file ready on your device.

## Layout on desktop vs mobile

- Desktop:
  You can keep **Tools** open while still seeing other moderation areas.
- Mobile:
  Open **Tools**, finish file actions there, then scroll to continue with other tasks.

## Upload and Manage Attachments

### Step-by-step

1. Open the file manager:
   - In **Tools** -> **Attachments**, click **Manage Files**.
2. Upload a file:
   - Optional: enter a label (for a clearer public name).
   - Choose the file.
   - Click **Upload Attachment**.
3. Check the uploaded entry:
   - If a label was entered, the list shows `Label (filename)`.
   - If no label was entered, the list shows the filename.
4. Remove outdated files:
   - Click **Delete** on the file you no longer need and confirm.

### What you should see

- New files appear in the attachment list immediately.
- Upload and delete actions update the list without leaving the page.
- The list in **Tools** stays aligned with the file manager dialog.

## Set or Clear the Current Document

Use this when you want attendees to focus on one specific file.

### Step-by-step

1. In the attachments list, click **Set as Current** on the file you want to present.
2. Confirm the same row now shows **Clear**.
3. When discussion moves on:
   - Click **Clear** to remove the current document, or
   - click **Set as Current** on another file to switch.

### What attendees should see

- After **Set as Current**:
  The live page shows document actions for opening/downloading the current file.
- After **Clear**:
  The live page no longer shows that current-document action block.

## If something goes wrong

- You cannot open file controls:
  Check that an agenda point is active, then open **Tools**.
- You cannot find attachments on the main moderation page:
  Attachments are managed in **Tools** via **Manage Files**.
- Uploaded file name looks unclear:
  Re-upload with a short, clear label and delete the unclear entry.
- Attendees report the wrong file:
  Verify which row has **Clear** (that row is currently selected), then switch if needed.
- Mobile workflow feels slow:
  Complete all file actions in one pass before switching to another section.
