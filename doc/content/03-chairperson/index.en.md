# Chairperson Guide

Use this section to run a meeting in a clear order, from preparation to live moderation tasks.

## Who this is for

Chairpersons and designated moderators who run committee meetings.

## Before you start

1. Confirm your meeting role allows moderation actions.
2. Decide your current phase:
   - meeting setup and activation
   - live moderation
   - specific task (agenda, attendees, speakers, files, votes, hand-off)

## Step-by-step

1. Start with [Committee Dashboard and Meeting Lifecycle](/docs/03-chairperson/01-committee-dashboard-and-meeting-lifecycle) to create, activate, and manage meeting sessions.
2. Continue with [Moderate Workspace Overview](/docs/03-chairperson/02-moderate-workspace-overview) to understand where each live control is located.
3. Use [Agenda Management and Import](/docs/03-chairperson/03-agenda-management-and-import) for manual agenda edits and controlled imports.
4. Use [Attendees Signup and Recovery](/docs/03-chairperson/04-attendees-signup-and-recovery) to manage participant entry and restore guest access.
5. Use [Speakers Moderator and Quotation](/docs/03-chairperson/05-speakers-moderator-and-quotation) to run the speaking queue and speaking rules.
6. Use [Attachments and Current Document](/docs/03-chairperson/06-attachments-and-current-document) to manage files and control what attendees see as the current document.
7. Use [Vote Moderation Open and Secret](/docs/03-chairperson/07-vote-moderation-open-and-secret) to create, open, close, and archive votes.
8. Use [Join QR and Access Hand-Off](/docs/03-chairperson/08-join-qr-and-access-hand-off) to onboard new guests and help returning guests re-enter quickly.

## What you should see

- A practical order for daily chairperson work.
- Less switching back and forth between unrelated guides.
- Faster recovery when a live meeting issue appears.

## If something goes wrong

- You are unsure which page to open:
  Start with the current meeting phase (setup, live moderation, or task-specific action).
- A live issue appears and timing is tight:
  Open the matching task guide directly from this index (attendees, speakers, files, votes, or hand-off).
