# Vote Moderation Open and Secret

![Vote lifecycle gif](../../assets/captures/app-vote-lifecycle-open-and-secret.en.light.desktop.gif)

Use this page to run meeting votes from start to finish without leaving moderation.

## Who this is for

Chairpersons who create, open, close, and finalize votes during a live meeting.

## Before you start

1. Open the meeting moderation page.
2. Activate the agenda point the vote belongs to.
3. Open the **Tools** tab and find **Votes**.
4. Decide whether you need an **Open** vote or a **Secret** vote.

## Layout on desktop vs mobile

- Desktop:
  You can keep **Tools** and **Votes** visible while monitoring the rest of moderation.
- Mobile:
  Open **Tools**, finish vote actions there, then scroll to other sections.

## Run an Open Vote

### Step-by-step

1. Create the draft:
   - Open **Create Vote**.
   - Enter a vote name.
   - Set visibility to **Open**.
   - Set min/max selections.
   - Keep at least two non-empty options.
   - Click **Create Draft Vote**.
2. Final check before opening:
   - Open the vote card and verify title, options, and limits.
   - Announce what attendees should select.
3. Start voting:
   - Click **Open Vote**.
4. Monitor submissions:
   - Watch the counters in the vote card (`Eligible`, `Casts`, `Counted Ballots`, `Outstanding`).
5. End voting:
   - Click **Close Vote**.
6. Finalize:
   - Review final tallies.
   - Click **Archive Vote** when the result is final.

### What attendees should see

- While open: attendees can choose options and click **Submit Open Ballot**.
- After close: results appear briefly on the live page, then disappear again.

## Run a Secret Vote

Use this when attendees should not reveal their selected option in public.

### Step-by-step

1. Create the draft:
   - In **Create Vote**, set visibility to **Secret**.
   - Fill name, options, and min/max selections.
   - Click **Create Draft Vote**.
2. Open vote:
   - Click **Open Vote**.
3. Register incoming casts:
   - In the vote card, use **Register Cast** for each attendee who submitted a ballot.
4. Count ballots:
   - Use **Count Ballot** to enter counted results.
5. Close vote:
   - Click **Close Vote**.
   - If counting is incomplete, the vote stays in counting mode.
6. Finish counting and close again if needed:
   - Enter remaining ballots, then close.
7. Archive:
   - Click **Archive Vote** after final confirmation.

### What to confirm

- Secret vote results stay blocked while counting is incomplete.
- The vote only fully closes after required counting is done.
- Receipt checks work only after the vote is closed.

## If something goes wrong

- You cannot create a vote:
  Check whether an agenda point is active.
- Create or update is rejected:
  Ensure name is filled, limits are valid, and at least two options are non-empty.
- A late attendee cannot vote:
  Only attendees already in the meeting when the vote opens can vote.
- A duplicate ballot is rejected:
  Keep one valid submission per attendee and continue.
- Secret vote will not close:
  Complete counting first, then close again.
- Mobile moderation is hard to track:
  Complete one vote phase at a time (create -> open -> monitor -> close -> archive).
