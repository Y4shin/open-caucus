# Public Verification

Use this section to verify vote receipts without moderator access.

## Who this is for

Participants and observers who want to validate published vote receipts.

## Before you start

1. Keep the browser/profile where receipts were stored.
2. Wait until the relevant vote is closed.

## Step-by-step

1. Open [Receipts Vault and Receipt Verification](/docs/05-public-verification/01-receipts-vault-and-receipt-verification).
2. Refresh the receipt list.
3. Verify each receipt you want to check.

## What you should see

- Clear success/error feedback for each receipt.
- Local receipt storage controls (`Refresh`, `Clear All`).

## If something goes wrong

- No receipts appear:
  Use the original browser profile where voting happened.
- Verification fails repeatedly:
  Retry after vote closure and check the receipt entry again.
