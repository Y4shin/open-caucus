# Hilfe, Suche und Dokumentnavigation

## Verhalten des Hilfe-Elements

Das Hilfe-Element unterstützt:

- Volltextsuche über `/docs/search`
- baumförmige Navigation mit nummerierter Sortierung
- Breadcrumb-/Pfadanzeige aus Verzeichnis-Index und Seitentiteln im Frontmatter

## Strukturregeln

- Jedes Verzeichnis benötigt `index.{en,de}.md`.
- Jede Seite benötigt `title-en` und `title-de` im Frontmatter.
- Fehlende Titel führen zu einem Startfehler der Anwendung.
