# Guides

Use this section to maintain and update the documentation set.

## Who this is for

Documentation maintainers.

## Step-by-step

1. Use [Capture Guide](/docs/guides/01-capture) when updating screenshots or GIFs.
2. Use [Document Authoring Standards](/docs/guides/02-document-authoring-standards) when writing or reviewing pages.

## What you should see

- Consistent capture quality and naming.
- Consistent page structure and metadata across docs.
